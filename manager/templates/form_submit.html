Command function String to send Description of the command
SET TRANSLATE distance_cm ST:[D] Order wall-e to move to the X,Y coordonate then rotate to the inputted angle
SET ROTATE rotation_deg SR:[R] Order wall-e to move to the X,Y coordonate then rotate to the inputted angle
SET HEAD Rotation SH:[R] Order wall-e to rotate the head to the inputted angle
SET SHOULDER (LEFT || RIGHT) Rotation SS[L || R]:[R] Order wall-e to rotate the inputted shoulder (left or right) to rotate to the inputted angle
SET ARM (LEFT||RIGHT) Rotation SA[L || R]:[R] Order wall-e to rotate the inputted arm (left or right) to rotate to the inputted angle
GET HEAD GH Get the actual angle of wall-e's head
GET SHOULDER (LEFT||RIGHT) GS[L || R] Get the actual angle of wall-e's inputted shoulder (left or right)
GET ARM (LEFT||RIGHT) GA[L || R] Get the actual angle of wall-e's inputted arm (left or right)
GET GYRO GG Get the data of the gyroscope
